Rekognize

The goal of the project is to develop an application that receives an image, uploads it to an AWS S3 Bucket, and triggers an AWS Lambda Function to analyze the faces detected in the image using Amazon Rekognition. Subsequently, ChatGPT will generate a news article about the individual in the image and display this article on a website.
